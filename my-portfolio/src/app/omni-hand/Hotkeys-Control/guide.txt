3. HOTKEY COMMANDS
-------------------
- [ F2 ] : OPENS AND CLOSE THE APP (Check bottom left for gray dot it indicates that it's running)

- [ ALT + X ] : TOGGLE VIEW (Hides/Shows the answer box)
- [ ALT + C ] : CLEAR (Deletes the current answer)

4. READ THE HEARTBEAT (Bottom-Left Dot)
---------------------------------------

- BLUE  : Scanning (Blinking)
- GREEN : Result Ready (answers will be is ready to read in bottom-right)
